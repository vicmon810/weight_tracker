## 项目目标
搭建一个 **深入嵌入的个人智能助手** 用于解释身体数据，生产记录观察，并在Ipone上作为系统能力存在。

### 核心
- 智能助手不只是聊天页面，而是系统解析能力
- 系统负责计算，LLM只负责解析
- 不作为任何的医疗诊断
